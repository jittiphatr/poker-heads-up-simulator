# Poker Heads-Up Simulator
A Python code that runs a simulation on a heads-up match for practice and to see how brutal the variance is in some of the hands. It runs completely randomly and prints the result in the terminal.

## How to run the code
Run the python file in your code editor or terminal:
```bash  
python main.py  
```
No input is required. The program will automatically deal the cards and print the result.
